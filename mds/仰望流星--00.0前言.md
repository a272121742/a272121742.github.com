##三部曲##

###构建项目###

```:DOS
$ meteor create app
```

###运行项目###

```:DOS
$ cd app
$ meteor
```

接着你将会看到命令行提示“**Meteor server running on: http://localhost:3000**”的字样，证明你的项目已经跑起来了。

###修改项目###

`Meteoer`是基于[NodeJS](http://nodejs.org)的，但他是热部署的，无论你怎么修改都不用重启服务器。你将只关心开发过程，保存后你也会实时的看到修改的效果。
